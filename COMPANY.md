### Fonts

**Primary Font:** Poppins (self-hosted)

| Weight | Usage |
|---|---|
| 300 (Light) | Body copy, default UI text |
| 400 (Regular) | General text |
| 500 (Medium) | Slightly emphasized text |
| 600 (Semibold) | Labels, pills, nav items |
| 700 (Bold) | h3, h4, feature labels |
| 800 (Extrabold) | h1, h2, prices, card headings |
| 900 (Black) | Rare heavy display use |

Fallback: `"Raleway", sans-serif`

### Typography Scale

| Element | Size | Line Height | Weight |
|---|---|---|---|
| h1 | 2.875rem (46px) | 3.45rem | 800 |
| h2 | 2.25rem (36px) | 2.75rem | 800 |
| h3 | 1.25rem (20px) | 1.925rem | 700 |
| h4 | 22px | 34px | 700 |
| Body / p / li | 17px | 32px | 300 |
| Display label | 17px | 20px | 600 |

**Pill / label style:** 13px, weight 600, uppercase, letter-spacing 0.04em
